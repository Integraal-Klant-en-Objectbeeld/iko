# Security Policy

## Supported Versions

The following versions of this project are currently supported with security updates:

| Version | Supported |
| ------- | --------- |
| 1.0.0   | ✅ Yes    |
| < 1.0.0 | ❌ No     |

Version **1.0.0** is the **first official release** of this project.  
Only the latest released version is supported with security fixes.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

- **Do not** open a public issue.
- Instead, contact the maintainers privately.

### How to report
- Send an email to: **security@ritense.com**  
- Include:
    - A description of the vulnerability
    - Steps to reproduce
    - Potential impact
    - Any suggested mitigations (if available)

### What to expect
- We will acknowledge receipt of your report as soon as possible.
- We will investigate and take appropriate action.
- If the issue is confirmed, a fix will be released for the supported version.

Thank you for helping keep this project secure.
